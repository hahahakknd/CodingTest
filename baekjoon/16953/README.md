A → B
============================
|시간 제한|메모리 제한|
|:---:|:---:|
|2초|512MB|

## 문제
-------
정수 A를 B로 바꾸려고 한다.</br>
가능한 연산은 다음과 같은 두 가지이다.</br>

- 2를 곱한다.
- 1을 수의 가장 오른쪽에 추가한다.

A를 B로 바꾸는데 필요한 연산의 최솟값을 구해보자.</br>

## 입력
-------
첫째 줄에 A, B (1 ≤ A < B ≤ 109)가 주어진다.</br>

## 출력
-------
A를 B로 바꾸는데 필요한 연산의 최솟값에 1을 더한 값을 출력한다.</br>
만들 수 없는 경우에는 -1을 출력한다.</br>

## 예제 1
-------
### 입력
```
2 162
```
### 출력
```
5
```
2 → 4 → 8 → 81 → 162</br>

## 예제 2
-------
### 입력
```
4 42
```
### 출력
```
-1
```

## 예제 3
-------
### 입력
```
100 40021
```
### 출력
```
5
```
100 → 200 → 2001 → 4002 → 40021</br>
