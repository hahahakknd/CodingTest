아스키 코드
============================
|시간 제한|메모리 제한|
|:---:|:---:|
|1초|256MB|

## 문제
-------
알파벳 소문자, 대문자, 숫자 0-9중 하나가 주어졌을 때, 주어진 글자의 아스키 코드값을 출력하는 프로그램을 작성하시오.</br>

## 입력
-------
알파벳 소문자, 대문자, 숫자 0-9 중 하나가 첫째 줄에 주어진다.</br>

## 출력
-------
입력으로 주어진 글자의 아스키 코드 값을 출력한다.</br>

## 예제 1
-------
### 입력
```
A
```
### 출력
```
65
```

## 예제 2
-------
### 입력
```
C
```
### 출력
```
67
```

## 예제 3
-------
### 입력
```
0
```
### 출력
```
48
```

## 예제 4
-------
### 입력
```
9
```
### 출력
```
57
```

## 예제 5
-------
### 입력
```
a
```
### 출력
```
97
```

## 예제 6
-------
### 입력
```
z
```
### 출력
```
112
```
